# 🌼CareKaro – We care and cure 🧑‍⚕️

### *Track 2 – Mumbai Hack 2025 (Healttech)*

CareKaro is a **full-stack MERN web platform** built to **revolutionize healthcare services in India**.  
We developed this project as part of **Track 2 at the Mumbai Hack 2025 (HealthTech category)**.

---

## 🚀 Project Overview

CareKaro is a **platform dedicated to helping India in various medical and healthcare domains**.  
Our website is designed to bridge the gap between patients and medical services using **modern AI-driven solutions** and a **responsive user interface**.

---

## ✨ Key Features

### 1. **Doctor Bot – AI-Powered Health Assistant**
- Built using **Botpress**.
- Analyzes symptoms and provides **guidance and recommendations**.
- Supports **multiple languages**, making it accessible across India.

### 2. **Medical Report Scanner**
- Uses **Google Gemini API** to analyze uploaded medical reports.
- Provides **detailed explanations** in **various Indian languages**.

### 3. **Help India – Medical Service Directory**
- Stores and serves data from **MongoDB**.
- Provides resources for **hospitals, blood banks, and NGOs**.
- Fully dynamic and easy to update.

### 4. **Medicine Availability & Delivery**
- Searches across multiple platforms to **check real-time medicine availability**.
- Helps users **find prescribed medicines faster**.

### 5. **Contact & About Pages with Email Service**
- Integrated **EmailJS** for instant email support.
- Used for **Contact Us** and **About Us** pages.

### 6. **Responsive UI & Health Updates**
- Built with **Tailwind CSS** and **React packages**.
- Contains:
  - **Navbar & Carousel** displaying **daily health updates**.
  - **Medical advertisement & awareness sections**.
- Fully **responsive** across all devices.

### 7. **Authentication & Security**
- **Clerk** for secure **authentication and user management**.
- **Arcjet** for **rate limiting** on critical routes and secure API endpoints.

### 8. **Map Integration**
- **Leaflet.js** for mapping and geolocation-based services.

### 9. **Visual & Design Resources**
- Logos and images sourced from **Freepik**, **Dribbble**, and other **free design platforms**.
- All names and demo content are **fictional & AI-generated** – *no personal sentiments harmed*.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, Tailwind CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **AI Services:** Botpress, Google Gemini  
- **Authentication:** Clerk  
- **Rate Limiting:** Arcjet  
- **Maps:** Leaflet.js  
- **Deployment:** Vercel  
- **Email Service:** EmailJS
